#  Disease Prediction Web App  

A simple web app that predicts the risk of *Diabetes, Heart Disease, and Parkinson’s* based on user health parameters like *BMI and Age*.  
Built using *HTML, CSS, and JavaScript*.  

##  Features  
- Predicts risk for multiple diseases  
- Simple and responsive UI  
- Runs directly in the browser (no extra setup needed)  

##  How to Run  
1. Download or clone this repository:  
   ```bash
   git clone https://github.com/Manyasethi07/Disease_Prediction.git
   ```
2. open the index.html file in your browser.
3. Start using the Disease_Prediction
